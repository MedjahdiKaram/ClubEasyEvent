Le plugin le plus simple pour gérer les événements, équipes, joueurs et lieux d’un club sportif.

== Description ==
- Gestion d’événements (dates, lieux, scores, RSVP).
- Gestion d’équipes et joueurs (numéros, postes, liaison compte WP).
- Shortcodes : [cee_schedule], [cee_roster].
- Intégration WooCommerce (paiement saison).
- i18n complète (fr, en, es), compatible WPML/Polylang.

== Installation ==
1. Uploadez le dossier du plugin dans wp-content/plugins.
2. Activez « Club Easy Event » dans Extensions.
3. Allez dans Club Easy Event → Paramètres pour configurer.

== Utilisation ==
- Créez vos Équipes et Joueurs.
- Créez vos Événements, affectez équipe et lieu.
- Affichez le calendrier : [cee_schedule team_id="123" season_id="456"].
- Affichez l’effectif : [cee_roster team_id="123"].

== Frequently Asked Questions ==
Q: Les menus n’affichent rien ?
R: Vérifiez que les sous-menus pointent vers edit.php?post_type=... comme dans ce plugin.
